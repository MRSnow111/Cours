О проекте Django Блог

Полнофункциональное блог-приложение для обучения

Цели проекта

Этот проект создан как учебный ресурс по веб-разработке на Django. Он охватывает все основные функции современного блога и следует лучшим практикам Django.

Реализованные функции

  • CRUD-операции
  • Аутентификация пользователей
  • Система комментариев
  • Теги через django-taggit
  • Поиск по записям
  • Пагинация
  • Профили пользователей
  • Сброс пароля

Архитектура

Проект построен по чистой архитектуре с двумя основными приложениями:

  • blog: Управляет всей функциональностью блога: записями, комментариями, тегами и поиском.
  • accounts: Отвечает за аутентификацию, регистрацию, профили и восстановление пароля.

Технологии

Python 3.10+ Django Bootstrap 5 SQLite django-taggit Pillow
